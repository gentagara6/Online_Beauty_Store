Online Beauty Store - Glow Beauty

Online Beauty Store është një faqe e-commerce për produkte bukurie e ndërtuar me HTML, CSS dhe JavaScript. Ky projekt shërben si një website demonstrues ku përdoruesit mund të shohin produkte, përshkrime, galeri dhe faqe kontakti.

---
 Përmbajtja e projektit

 Struktura kryesore e projektit përfshin:

- `index.html` – Homepage
- `shop.html` – Faqja e dyqanit
- `sproduct.html` – Detajet e produktit
- `about.html`, `blog.html`, `contact.html` – Faqet informuese
- Folders:
  - `css/` – Stilet CSS
  - `js/` – Skedarët JavaScript
  - `img/` – Imazhe të përdorura në faqe

---
 Teknologjitë e përdorura

Projekti është ndërtuar me:

- HTML5 – Strukturë
- CSS3 – Dizajn & layout
- JavaScript – Ndërveprimet bazë, kryesisht librarine jQuery
